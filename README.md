# 使用文档
将项目克隆之本地，使用 Dev-C++ 打开项目文件 `/prjdev/CodeEditor.dev` 即可编译运行，
亦可以对项目进行编辑

有关本项目的更多信息，如选题背景，需求分析，程序开发设计，
部署运行说明以及团队合作记录，**或者只是看看我们做的什么样子**，请参见 [互评文档](docs/lyther-documentation.pdf)
